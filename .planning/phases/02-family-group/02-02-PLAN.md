### Task 4: Create FamilyDrawer.vue component
**File:** `frontend-ia/src/components/FamilyDrawer.vue`
- Use `PrimeVue Sidebar` (position="right", :visible.sync, class="family-drawer")
- Accept props: `visible` (Boolean, v-model), `family` (Object or null)
- Emit events: `update:visible`, `family-action` (payload: { action, data })
- Internal state: `drawerState` (enum: 'no-group', 'create-form', 'join-form', 'has-group')
- Internal data: `newGroupName`, `joinCode`, `loading`, `error`
- **State 'no-group':** icon pi-users (48px, #4b5563), text "Você não está em nenhum grupo...", button "Criar Grupo Familiar" (primary green), divider "ou", button "Entrar com Código" (outline green)
- **State 'create-form':** input "Nome do Grupo *", button "Criar Grupo" (green), link "Cancelar" (gray)
- **State 'join-form':** input "Código de Convite *" (maxlength 6, auto-uppercase, placeholder "Ex: A3B7K9"), button "Entrar no Grupo" (green), link "Cancelar"
- **State 'has-group':** show family name (bold), code section (monospace text + "Copiar" small button), expiration text, members list (scrollable), action buttons
- Members list: each member row with avatar (computed color + initial), name, badges "Admin" (green bg) and/or "Você" (gray bg)
- Admin actions (conditional): "Expulsar" icon button (red, sm) on each member row (except self), "Gerar Novo Código" button, "Excluir Grupo" button (danger outlined)
- Member actions: "Sair do Grupo" button (danger outlined)
- Use `ConfirmDialog` PrimeVue for: sair do grupo, excluir grupo, expulsar membro
- Use `Toast` PrimeVue for: success/error feedback (copied code, joined group, left group, etc.)
- **Width:** `w-96` desktop, `w-full` mobile (media query or Sidebar width prop)

## Component Props & Events

### FamilyDrawer.vue
```
Props:
  - visible: Boolean (required, v-model)
  - family: Object | null (family data or null)

Emits:
  - update:visible(Boolean)
  - family-action({ action: 'created'|'joined'|'left'|'deleted', data?: any })

State machine:
  no-group → create-form (click Criar)
  no-group → join-form (click Entrar)
  create-form → no-group (click Cancelar)
  create-form → has-group (submit success)
  join-form → no-group (click Cancelar)
  join-form → has-group (submit success)
  has-group → no-group (left/deleted family)
```
