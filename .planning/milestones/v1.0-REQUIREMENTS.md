# Requirements Archive: Sem Aperreio — v1.0 MVP

**Archived:** 2026-04-30
**Status:** All v1 requirements implemented and verified

---

## v1 Requirements (Completed)

### Authentication

- [x] **AUTH-01**: Usuário pode criar conta com email, nome e senha
- [x] **AUTH-02**: Usuário pode fazer login e receber token JWT (access + refresh)
- [x] **AUTH-03**: Token JWT é enviado em toda requisição à API
- [x] **AUTH-04**: Logout invalida/limpa token no frontend
- [x] **AUTH-05**: Apenas usuário autenticado pode criar/editar/excluir seus próprios gastos

### Family Group

- [x] **FAM-01**: Usuário pode criar um "grupo familiar" (ex: "Família Silva")
- [x] **FAM-02**: Usuário pode convidar outro usuário para o grupo via código/link
- [x] **FAM-03**: Membros do mesmo grupo veem todos os gastos do grupo
- [x] **FAM-04**: Gastos são vinculados ao usuário que os criou, mas visíveis ao grupo

### Gastos (refatoração do existente)

- [x] **GAST-01**: CRUD de gastos com autenticação (reutilizar modelo existente + user FK)
- [x] **GAST-02**: Filtro por categoria funciona com autenticação
- [x] **GAST-03**: Filtro por período (mês/ano) funciona com autenticação
- [x] **GAST-04**: Paginação com metadados (count, next, previous)

### Dashboard

- [x] **DASH-01**: Exibe total do mês atual vs. mês anterior
- [x] **DASH-02**: Gráfico de pizza: gastos por categoria (mês atual)
- [x] **DASH-03**: Gráfico de linha: evolução mensal (últimos 12 meses)
- [x] **DASH-04**: Ranking das categorias mais onerosas
- [x] **DASH-05**: Média diária e maior gasto do mês
- [x] **DASH-06**: Dashboard filtra por mês/ano selecionado pelo usuário

### Previsão (ML)

- [x] **ML-01**: Endpoint de previsão usa dados reais do usuário/grupo para treinar modelo
- [x] **ML-02**: Modelo considera categoria como feature adicional
- [x] **ML-03**: Retorna previsão de gasto total para o próximo mês

### Exportação

- [x] **EXP-01**: Exportar gastos filtrados para CSV
- [x] **EXP-02**: Exportar gastos filtrados para Excel (XLSX)
